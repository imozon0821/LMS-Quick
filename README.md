# LMS Quick

金沢大学生のためのChrome拡張機能です。LMS・アカンサスポータルなどをより便利に使えるようにします。

## 主な機能

- 学務情報サービス（教務システム）から時間割を自動で一括登録
- 登録した時間割をボタン一つで表示し、LMSへ直接ジャンプ
- LMSの課題を自動・一括スキャンして締切をアラート表示
- 締切が近い課題を「緊急」「注意」で色分け管理
- 課題の完了チェック機能

## 動作環境

- PC版 Google Chrome
- PC版 Microsoft Edge

## インストール方法

配布しているZIPファイルをダウンロードし、Chromeのデベロッパーモードから手動でインストールしてください。
詳しい手順はユーザーガイドをご覧ください。

## プライバシー

取得・保存するデータはすべてお使いのPC内にのみ保存されます。
外部サーバーへのデータ送信は一切行いません。

## 使用している権限

| 権限 | 用途 |
|------|------|
| `storage` | 時間割・課題データをローカルに保存するため |
| `tabs` | WebClassのタブを通じて課題スキャンを実行するため |
| `host_permissions` | eduweb・acanthus・WebClassへのアクセスのため |
